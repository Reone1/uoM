   3. 최근 공부한 내용이 반영된다면 좋을 것 같다.
   4. 내가 다루는 언어 타입이나 github 활동 내역을 반영

   어떻게 보면 포트폴리오 페이지를 하나 만든다는 생각이 들었다.
